# Republic C.A. — Brand Assets

Repositório oficial de assets de marca da **Republic C.A. Pizza & Chopp** — São José dos Campos, SP.

> Qualquer dúvida sobre aplicação: douglas@republicca.com.br  
> A marca Republic é registrada no INPI. Reprodução ou uso não autorizado configura infração à Lei nº 9.279/96.

## Logos Principais — Variações

Cada versão (principal, reduzida, horizontal) está disponível em **6 variações cromáticas**:

| Arquivo | Fundo | Uso |
|---|---|---|
| `*-colorido-transparente.svg` | Transparente | Digital geral, sobre fundos claros ou escuros |
| `*-colorido-navy.svg` | Navy `#1B4257` | Dark mode, materiais digitais |
| `*-colorido-preto.svg` | Preto `#000000` | Merch, impressão escura |
| `*-pb-branco.svg` | Branco `#FFFFFF` | Documentos, impressão simples |
| `*-pb-navy.svg` | Navy `#1B4257` | Dark mode sem cor |
| `*-pb-preto.svg` | Preto `#000000` | Versão máxima de contraste |

## Convenções de Nomenclatura

```
[versao]-[variacao].[ext]

Exemplos:
  logo-principal-colorido-transparente.svg
  logo-reduzido-pb-navy.png
  mascote-sem-fundo.svg
  mascote-com-fundo-reduzido.svg
```

Formatos preferidos: `.svg` para vetores, `.png` (fundo transparente) para raster.
