# Republic CA — Brand Assets

Repositório centralizado de assets de marca da Republic CA. Use este repositório para acessar logos, paletas, tipografia e guidelines atualizados.

## Estrutura

```
republic-brand/
├── logo/
│   ├── primary/        # Logo principal (PNG, SVG, PDF, AI)
│   ├── secondary/      # Versões secundárias
│   └── variations/     # Versões monocromática, negativa, etc.
├── colors/             # Paleta de cores (swatches, referências HEX/CMYK/Pantone)
├── typography/         # Fontes e guidelines tipográficos
├── icons/              # Ícones e elementos gráficos
└── guidelines/         # Manual de marca (PDF)
```

## Uso

- Sempre use os arquivos desta pasta como referência oficial
- Para peças digitais: prefira os arquivos `.svg` ou `.png` em alta resolução
- Para impressão: use os arquivos `.ai` ou `.pdf`
- Dúvidas sobre aplicação da marca: consulte `guidelines/`

## Atualização

Qualquer alteração nos assets deve ser aprovada antes de subir neste repositório.
